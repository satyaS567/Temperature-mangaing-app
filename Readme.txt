It is a simple weekly temperature management application that was created using Python and Python filehandly. Menu-driven applications and dictionaries are used to display the day's smart temperature as well as common temperatures for 7 days of the week. Additionally, you can get temperature data below 0 degrees Celsius in which day the temperature is also equal to zero. The exterior temperature sensor can be used to get mesurement of equivalent temperate values under different conditions.
There no need to any tool to make this app only you can use vscode or jupyter notebook and tempature data.
